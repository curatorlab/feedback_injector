# FeedbackInjector

Context : Some curating platforms web interfaces restrict the standard paste functionality into feedback fields, making it difficult to quickly insert an appropriate review.
FeedbackInjector provides a simple keyboard shortcut allowing to instantly inject the text copied in clipboard into text fields, helping you maintain a fast and efficient review workflow.

## Features

* Instant clipboard text injection : with the Ctrl + Q keyboard command
* Custom keyboard shortcut support
* Lightweight and fast
* Works alongside curator platforms and web browsers
